# e-plantShopping - Paradise Nursery

Welcome to **e-plantShopping**, the official repository for Paradise Nursery. This is a React-based application designed to provide a seamless plant shopping experience.

## Features
- **Landing Page:** A beautiful entry point to our nursery.
- **Product Listing:** Browse through 3 categories with 6 unique plants each.
- **Shopping Cart:** Add, remove, and adjust quantities of your favorite plants.
- **State Management:** Powered by Redux Toolkit.

## Installation
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm start` to launch the application.
